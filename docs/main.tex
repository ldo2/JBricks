\def\abstract{
Пособие содержит изложение ряда теоретических понятий учебного курса
<<Методы хранения и обработки информации>>, краткое описание языка 
UML и общих принципов проектирования объектно-ориентированных систем,
несколько вариантов заданий на модификацию эталонного проекта и подробный
разбор одного из них.

В результате выполнения курсовой работы студенты получают практические
навыки работы с небольшими объектно-ориентированными проектами и опыт
модификации существующего программного кода. Они знакомятся с языком
моделирования UML и углубляют знания языка программирования Java.
}

\def\intro{
Методическое пособие подготовлено автором на основе опыта чтения курсов 
<<Методы хранения и обработки информации>> и <<Практикум на ЭВМ>> студентам
Московского государственного индустриального университета (направление
510200 и специальности 220400 и 351500).

Курсовая работа заключается в выполнении одного из заданий на модификацию
описанного в пособии проекта --- игры <<Кирпичики>>. Этот проект был выбран
для обучения студентов концепциям объектно-ориентированного
проектирования и программирования на языке Java потому, что он достаточно прост
и допускает различные расширения. Текст эталонного проекта
на языке Java, содержащий многочисленные комментарии, выдаётся студентам 
вместе с заданием.

При разработке проекта и подготовке методического пособия было использовано
только свободно распространяемое программное обеспечение. Его же рекомендуется
применять и при выполнении курсовой работы.
}
